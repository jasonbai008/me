# Me - 个人移动端展示页

一个简洁、美观的个人移动端自展示页面，采用响应式设计，专为移动端访问优化。

## 🌟 项目简介

本项目是一个基于原生 HTML/CSS/JS 开发的个人名片页。它通过简约而富有现代感的设计，展示了个人的基本信息、职业生涯、兴趣爱好以及联系方式。

## ✨ 功能特点

- **响应式设计**：针对移动端优化，完美适配各种屏幕尺寸。
- **动感交互**：
  - 头像掉落入场动画。
  - 点击头像可放大查看原图。
  - 交互式微信联系弹窗，支持一键复制微信号。
- **结构化内容**：
  - **基本信息**：清晰展示个人背景资料。
  - **工作经历**：采用时间轴设计，突出最近的职业动态。
  - **兴趣爱好**：标签化展示，直观了解个人生活。
  - **联系方式**：集成个人主页、博客及社交账号链接。

## 🛠️ 技术栈

- **HTML5**：语义化结构设计。
- **CSS3**：
  - 使用 **Flexbox** 和 **Grid** 进行页面布局。
  - 纯 CSS 实现动画效果（`@keyframes`）。
  - 使用 CSS 变量和渐变色提升视觉体验。
- **Vanilla JavaScript**：原生 JS 处理 DOM 交互，无需第三方依赖，加载迅速。

## 📂 项目结构

```text
me/
├── assets/          # 静态资源文件（头像、图片等）
├── index.html       # 核心 HTML 结构与样式逻辑
└── README.md        # 项目说明文档
```

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/your-username/me.git
   ```

2. **本地运行**
   由于是纯静态页面，您可以直接在浏览器中打开 `index.html` 即可预览效果。

3. **个性化定制**
   - 修改 `index.html` 中的文字内容。
   - 替换 `assets/` 目录下的图片。
   - 在 CSS 部分调整主题颜色（`linear-gradient`）。
